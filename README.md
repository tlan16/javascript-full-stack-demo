# javascript-full-stack-demo
![](https://github.com/tlan16/javascript-full-stack-demo/workflows/CI/badge.svg)

This is a demo JavaScript Full-stack project

## To Run

```bash
docker-compose pull
docker-compose build
docker-composer up
```

Open http://127.0.0.1:8888

## Tech stack

### Frontend

- react
- react-router
- jest
- webpack

### Backend

- node
- express
- mongoose
- jest
